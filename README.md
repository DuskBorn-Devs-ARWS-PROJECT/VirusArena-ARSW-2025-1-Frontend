# VirusArena-ARSW-2025-1-Frontend
```
│── static
│   ├── HostLobby.html      # Pantalla para que el host configure la partida
│   ├── Login.html          # Pantalla de ingreso de nombre de usuario
│   ├── MainLobby.html      # Lobby principal donde se puede crear o unirse a partida
│   ├── WaitingLobby.html   # Sala de espera antes de que inicie la partida
│   ├── Game.html           # Interfaz principal del juego
│   ├── GameOver.html       # Pantalla de fin de partida
│
│── js
│   ├── gameClient.js       # Maneja la conexión WebSocket con el servidor
│   ├── lobbyManager.js     # Lógica para crear y unirse a partidas
│   ├── map.js              # Renderiza el mapa en un `<canvas>`
│   ├── player.js           # Controla el movimiento y estado del jugador
│   ├── ui.js               # Maneja la interfaz, tiempo y chat del juego
│
│── css
│   ├── styles.css          # Estilos y diseño del juego
```
